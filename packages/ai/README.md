# AI Core

Noyau IA indépendant de FlowPilot. Il coordonne les modèles, prompts, mémoires, plans, validations et mesures sans dépendre d’un fournisseur, d’Express ou de Prisma.

Les adaptateurs réseau vers OpenAI, Anthropic, Gemini, Mistral et DeepSeek seront branchés sur `AiProvider`. Ollama est déclaré comme extension future. Aucun SDK fournisseur n’est une dépendance du noyau.

Voir [la documentation complète](../../docs/ai-core.md).
